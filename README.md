# Library-Management-System
Library Management System using Java (Spring boot), postgress SQL database and JWT Security

Description: The Library Management System is a Spring Boot application that allows users to manage books, patrons, and borrowing records efficiently. It provides APIs for adding, updating, retrieving, and deleting books, as well as managing librarians and borrowers.

Features • Book Management (Add, Update, Delete, Retrieve) • Patron Management • Borrowing Records • API Logging and Exception Handling using Spring AOP • Swagger API Documentation

Tech Stack • Spring Boot (REST API) • JWT Authentication & Rate Limiter (Security) • Spring Data JPA (Hibernate ORM) • PostGreSQL (Database) • Swagger (API Documentation) • Spring AOP (Logging) • Ehcache (Caching) • Transactional for to ensure data integrity during critical operations. • JUnit & Mockito (Unit Testing)

With clean code and handling Exception and input validation, This project done Successfully.

Prerequisites To Run the project you should have: • Java 17+ • Maven 3.8+ • IDE (IntelliJ, Eclipse, VS Code) • PostgreSQL (I used docker) Just run the project and go to: http://localhost:8080/swagger-ui/index.html#/
